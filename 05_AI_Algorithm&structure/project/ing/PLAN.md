# 주제: AI 시험 감독관
## 1. 핵심 기능 (Core Features)
* 저희 AI 온라인 시험 감독 프로그램은 시험의 공정성과 신뢰성을 확보하기 위해 다음과 같은 핵심 기능을 제공합니다.
```
    • 1.1. 신원 확인 (Identity Verification)
        ◦ 시험 응시자의 신원을 정확하게 확인하여 대리 시험 또는 본인 불일치로 인한 부정행위를 사전에 방지합니다.
    • 1.2. 부정행위 감지 (Cheating Detection)
        ◦ 시험 진행 중 발생하는 다양한 형태의 부정행위 징후를 실시간으로 감지하고 분석합니다.
    • 1.3. 알림 및 기록 (Notification & Logging)
        ◦ 감지된 부정행위에 대한 즉각적인 알림을 발생시키고, 관련 이벤트 및 데이터를 상세하게 기록하여 관리자가 추후 검토 및 증거 자료로 활용할 수 있도록 합니다.
    • 1.4. 웹 연동 (Web Integration) - 추가 기능
        ◦ 시스템을 웹 기반 플랫폼과 연동하여 시험 관리의 편의성을 높이고, 응시자 및 관리자에게 효율적인 인터페이스를 제공합니다.
```
## 2. 필요 기술 (Required Technologies)
* 이 프로젝트의 성공적인 구현을 위해 다음과 같은 핵심 기술들이 활용될 예정입니다.
```
    • 2.1. OpenCV
        ◦ 오픈 소스 컴퓨터 비전 라이브러리로, 실시간 비디오 스트림 처리, 이미지 분석, 객체 감지 등 영상 데이터 처리의 기반 기술로 사용됩니다.
    • 2.2. Eye Tracking (시선 추적)
        ◦ 응시자의 눈동자 움직임을 정밀하게 추적하여 시선의 비정상적인 이동 (예: 다른 곳을 응시하는 행위)을 감지하는 데 활용됩니다.
    • 2.3. Face Recognition (얼굴 인식)
        ◦ 응시자의 얼굴을 인식하여 본인 인증을 수행하고, 동시에 여러 얼굴이 감지되거나 등록되지 않은 얼굴이 나타나는 경우를 식별하는 데 사용됩니다.
```
## 3. 활용 사례 (Use Case)
* 본 AI 온라인 시험 감독 프로그램의 일반적인 활용 흐름은 다음과 같습니다.
```
    • 3.1. 웹 페이지 접속 및 로그인
        ◦ 응시자는 지정된 웹 페이지에 접속하여 개인 계정으로 로그인합니다. (시스템 연동 시)
    • 3.2. 얼굴 인식을 통한 본인 인증 및 시험 시작
        ◦ 시스템은 응시자의 웹캠을 통해 실시간으로 얼굴을 인식합니다.
        ◦ 인식된 얼굴이 사전에 등록된 응시자의 얼굴 정보와 일치하는지 확인합니다.
        ◦ 본인 인증이 성공적으로 완료되면 시험이 시작됩니다.
    • 3.3. 시험 진행 중 AI 기반 부정행위 실시간 감지
        ◦ 시험이 진행되는 동안, AI 모델은 응시자의 행동을 지속적으로 모니터링하며 다음과 같은 부정행위 징후를 실시간으로 감지합니다.
            ▪ 3.3.1. 시선 추적 (Eye Tracking): 응시자의 눈동자가 일정 기준치 이상으로 좌우 또는 다른 방향으로 빈번하게 움직이는지 감지합니다.
            ▪ 3.3.2. 머리 움직임 감지: 응시자의 고개가 일정 기준치 이상으로 옆으로 돌아가거나, 화면 밖을 응시하는 등 비정상적인 움직임을 보이는지 감지합니다.
            ▪ 3.3.3. 복수 인원 감지: 웹캠 화면에 응시자 외에 두 명 이상의 인원이 동시에 감지되는지 확인합니다.
            ▪ 3.3.4. 자리 이탈 감지: 응시자가 시험 중 일정 시간 이상 웹캠 화면에서 벗어나 자리를 이탈하는지 감지합니다.
    • 3.4. 부정행위 감지 시 내부 카운트 증가
        ◦ 각 부정행위 징후가 감지될 때마다 해당 응시자의 내부 부정행위 카운트가 증가합니다.
```
